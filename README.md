# ⚖️ Adalet Peşinde

Avukat olup müvekkillerini savunduğun, kısa ve eğlenceli davalarla seviye atladığın mobil oyun.

**Oyna:** https://elethroth.github.io/adalet-pesinde/

- 12 dava, her biri ~2 dakika
- Kanıta dayalı savunma seç, hâkimi ikna et
- Yıldız topla, rütbe atla: Stajyer Avukat → Adalet Efsanesi
- Telefona kurulabilir (Safari/Chrome → "Ana Ekrana Ekle"), çevrimdışı çalışır

Tek HTML dosyası + PWA. Ses efektleri WebAudio ile, görseller SVG ile kodun içinde üretilir; harici bağımlılık yoktur.
